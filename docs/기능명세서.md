# 기능 명세서
> KoSI 프로젝트 기능 명세서

## Frontend
| 구분 | 기능 | 설명 |
|---|---|---|
| 메인 | 프로젝트 소개 | 프로젝트 개요 표시 |
| 질문 입력 | 질문 등록 | 원본 질문 입력 |
| 모델 선택 | GPT/Claude/Gemini 선택 | 평가 대상 선택 |
| 평가 실행 | 실행 | 평가 요청 |
| 결과 조회 | 대시보드 | 모델별 결과 시각화 |
| 이력 | 실행 이력 | 이전 평가 조회 |

## Backend
| 구분 | 기능 | 설명 |
|---|---|---|
| API | 평가 실행 API | 평가 요청 처리 |
| 데이터셋 | CRUD | 질문 등록·수정·삭제 |
| AI 연동 | GPT/Claude/Gemini | 모델 호출 |
| 평가 | BERTScore | 의미 유사도 계산 |
| 평가 | LLM-as-Judge | 답변 일관성 평가 |
| 평가 | Consistency Score | 종합 점수 계산 |
| DB | 결과 저장 | 평가 결과 저장 |
| 리포트 | CSV/JSON | 결과 다운로드 |

## 데이터셋
- 원본 질문
- 패러프레이즈 질문
- 의미 동치 검증
- 카테고리 관리

## 평가 지표
- BERTScore
- Sentence Embedding Cosine Similarity
- LLM-as-Judge
- Consistency Score
- Framing Sensitivity Score
